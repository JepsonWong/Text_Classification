煤样工业分析温控系统的设计
秦海峰　秦瑞杰
　　摘　要　介绍一种用于煤样分析的温度控制系统。硬件主要由热电偶专用放大器、8031单片机及其数字调功电路组成，软件采用了智能产生式控制算法。应用结果表明，仪器的各项指标均满足国标GB212-77。
　　关键词　煤样分析　温度控制　流程曲线　控制算法
Design of Temperature Control System for Coal-sample Industrial Analysis
Qin Haifeng,Qin Ruijie
　　Abstract　A temperature control system for coal-sample analysis is introduced in this paper.The hardware is mainly composed of thermocoouple special amplifier,8031 chip microprocessor and its digital power regulating circuit.The software takes intelligence production control algorithm.The practical application shows that all the indexes of the system fully meet the needs of GB212-77.
　　Key words　Coal-sample analysis,Temperature control,Process curve,Control algorithm
　　在一些用煤量大的的单位，如热电厂等，为了了解煤的质量，往往要计算煤的固定炭。这需要在一定的温度下对煤进行灼烧，然后称重，测量出煤的水分、灰分和挥发分，进而计算出煤的固定炭来实现的。如缓慢灰化检测法，就是先使煤升温到500 °C，使其恒温30 min后升温到815 °C，再恒温60 min，然后取出煤样称重。之后继续在815 °C环境下灼绕，每隔20 min取出称重，直到重量变化小于0.001 g为止。诸如此类的检测实验，关键问题是测、控温的问题。如果完全用人工进行过程控制，不仅难以保证升温、恒温的准确性，而且工作人员在长时间工作中容易因疲劳而造成不应的的失误。针对这种情况，我们以单片机为核心设计了煤样工业分析方法的温度控制系统。它具有自动化程度高、测控温精度高、提示信息丰富的特点。整个系统的设计和制做符合国家标准GB212-77《煤的工业分析方法》[1]，适用于褐煤、烟煤、无烟煤的水分、灰分、挥发分的测定。
1　系统电路构成及工作原理
　　系统结构框图如图1所示，由热电偶测量电阻炉内的温度，并转换成毫伏级的电压信号，经过冷端补偿，非线性校正及放大，送给A／D转换器。单片机读到数值后，变换成对应的温度值显示出来，同时将该温度值与流程要求的设定值进行比较。根据其偏差值的大小，采用人工智能领域中的产生式控制算法控制输出通道的控制输出量，也就是控制双向晶闸管的导通和截止时间，这样就控制了电热炉输入功率的大小，从而控制了电炉的温度，使其逐渐趋于设定值且达到平衡。




图1系统结构框图
　　有6个键分别对应6个流程，若想测定哪一项，只需按下对应的键，就会按既定的流程自动进行下去。其间，液晶显示会提示工作于哪一个流程，当前流程进行到哪一步以及所用的时间，下一步应干什么等。过程结束或温度异常，系统会给出声光提示，使整个测定工作变得十分简单。
2　各主要模块的实现
　　下面简单介绍一下本系统所包括的各主要模块。
2.1　单片机系统
　　本系统选用8031单片机为控制核心，扩展了一片程序存储器2764、一片数据存储器6116和一片并行接口芯片8255。
2.2　输入通道
　　按流程要求最高温度为1100 °C，故而我们选择了K型热电偶(量程为1300 °C)。但热电偶存在非线性及冷端补偿的问题，通常方法下使用很不方便，为此，我们选用了与K型热电偶对应的热偶放大器AD595[2]，它本身具有冷端温度补偿和线性化的功能。它的实际输入输出关系为：
Vo＝247.3(Vth＋0.011)
其中，Vo为AD595的输出电压，单位为mV；Vth为热电偶的输出电压，单位为mV。再结合K型热电偶的分度表，我们将会得到AD595的输出为10 mV／°C，可见它是线性输出，这使得硬件电路和软件编程都变得比较简单。A／D转换器我们选择了四位半双积分型的ICL7135，它具有抗干扰性强和精度高的特点，输入满度为±1.9999 V，把AD595的输出经过适当分压和滤波，送到ICL7135的输入端，电路如图2所示。ICL7135的输出是通过并行口8255读入的。


图2　输入电路
通常测温中，要求AD595置于热电偶冷端处，使之感受与测温热电偶冷端相同的温度，以达到正确的冷端温度补偿目的。可实际上，AD595往往只能置于温控箱中而远离热电偶冷端，为此我们选用了配套的补偿导线，将测温热电偶冷端延长到测控箱内。
2.3　输出通道
　　输出通道用来实现数字调功，以调节加入负载的功率而达到调节温度的目的。在电力电子领域，晶闸管及触发电路在调压、调功及自动化控制技术中有着广泛的应用。但为了解决触发电路和主回路的同步与隔离而使整个电路过于复杂，若采用同步变压器则会使体积庞大。美国MOTOROLA公司生产的光耦晶闸管MOC3061内部具有过零检测电路，可以很好地解决同步与隔离问题。由其组成的触发电路非常简单，如图3所示。


图3　输出电路
　　图中Rin为输入限流电阻，R为输出限流电阻，RG为晶闸管的门极电阻，RS和CS构成阻容吸收回路。

2.5　音乐提示
　　当定时时间到时，不仅液晶显示给出文字提示，还应给出声音提示。为了不与数据采集发生冲突而且要悦耳，我们选择了市场上非常便宜的音乐片，它体积小，信息量大，容易接口。用继电器控制它的不同的触发点，可产生不同的音乐，这样易于识辨不同的事件。因为只要瞬间触发一下即可，且能自动停止，所以不会影响采样率。
2.6　键盘及显示
　　由于本系统所用按键不多，故我们采用了独立式按键。它通过P1口来实现，具有与外中断INT1电路配置灵活，软件结构简单的特点。因为数据采集采用的是查询方式，故键盘采用了中断方式，电路图略，可参考文献[5]。
　　为了方便用户操作，我们采用了两行24字符的液晶显示模块，它是将LCD控制器、RAM、ROM和LCD显示器用PCB连接到一起构成的，与单片机接口十分方便，接口电路如图5所示。

3　控制算法的实现
　　我们先来介绍一下通断时间比控制，它是控制算法实现的基础。通断时间比控制是将一个基本的时间周期分为集中导通时间和断开时间两部分，导通时间在整个基本周期中所占的比例称为导通率。控制通断时间既是控制导通率，即让导通率的大小与控制器的输出量成正比。在计算机控制系统中，每经历一次固定的时间，控制器根据误差信号的大小和方向以及所采用的控制算法进行一次逻辑判断和数字运算，之后输出一个新的控制量并保持到下一个周期的开始，这段时间称为控制周期。系统调试时需要根据对象的特性的类型和滞后时间对控制周期进行“在线”整定。对于温度量，推荐值为15～20 s左右。一般说来，控制周期长于基本周期。在一个控制周期内，应让基本周期内的控制输出量完整的重复执行，即保持控制周期是基本周期的整数倍关系，这样才能保证执行器忠实地执行调节器的输出命令。
　　下面介绍我们所采用的智能产生式控制算法。我们都有过这样的亲身体验：当我们骑车赶向某一地点时，因为刚开始离目的地很远，我们会全力加速前进，当接近这一地点时，我们就必须减速前行，直到车速为零并停在目的地。产生式控制算法描述的就是这样一种思想，其规则的基本形式为：
IF(条件)　　　THEN(操作或结论)
　　实现该算法的具体方法是：单片机根据系统给定值yr与控制对象输出y(t)采样时刻的值y(KT)之偏差的大小，将其划分成若干控制局势Si(i＝1，2，…，n)。如图6所示，在每个控制局势下，都有与之对应的控制规则Ri。其相应的搜索规则分别为：

IFe1≤yr－y(KT) THEN S1
IFe2≤yr－y(KT)＜e1 THEN S2
IFe3≤yr－y(KT)＜e2 THEN S3
IF0≤yr－y(KT)＜e3 THEN S4
IF－e3≤yr－y(KT)＜0 THEN S5
IF－e4≤yr－y(KT)＜－e3THEN S6
IFyr－y(KT)＜－e4 THEN S7
IFy(KT)∈SiTHEN Ri


图6　控制局势划分
　　其中S1为大偏差情况，S2、S3为过渡情况，S4、C5为允许误差情况，S6为超调情况，S7为大超调报警情况。
　　对于每一个控制规则Ri，单片机输出相对应的输出量Ucj(j＝1，2，…，m)来控制数字调功器。而控制规则的实现是基于对系统的输出变化量△y(KT)判断，其中：
△y(KT)＝y(KT)－y(K－1)T
　　对应于图6所示的控制局势划分，其控制规则分别为
　IF　y(KT)∈S1 　THEN 　 u(KT)＝uc max
　IF y(KT)∈S2 THEN
　　{ IF △y(KT)＜0　THEN u(KT)＝uc max
　ELSE　 IF　 0≤△y(KT)＜c1　 THEN u(KT)＝uc1
　ELSE　 IF　 c1＜△y(KT) 　 THEN u(KT)＝uc2
　　}
IF　 y(KT)∈S3　 THEN
　　{ IF　 △y(KT)＜0 　 THEN　 u(KT)＝uc3
　ELSE　 IF　 0≤△y(KT)＜c2　 THEN　 u(KT)＝uc4
　ELSE　 IF　 c1＜△y(KT)　 THEN　 u(KT)＝uc5
　　}
　IF　 y(KT)∈S4　 THEN
{ IF　 △y(KT)≥0 　 THEN　 u(KT)＝ucs
　　ELSE　 IF　 △y(KT)＜0　 THEN　 u(KT)＝uc6
　　}
IF　 y(KT)∈S5　 THEN
　　{ IF　 △y(KT)≤0 　 THEN　 u(KT)＝ucs
　　ELSE　 IF　 △y(KT)＞0　 THEN　 u(KT)＝uc7
　　}
IF 　y(KT)∈S6 　 THEN
　　{ IF　 △y(KT)＜0 　 THEN 　 u(KT)＝uc8
　　ELSE　 IF 　△y(KT)≥0　THEN　 u(KT)＝0
　　}
IF　 y(KT)∈S7　 THEN　 u(KT)＝0
　　在上述控制规则中，控制量ucj的关系为ucmax＞uc1＞uc2＞uc3＞uc4＞uc5＞uc6＞ucs＞uc7＞uc8＞0。其中ucmax是在整个控制周期T内使主回路晶闸管导通的控制量，ucs是使炉温达到平衡，即yr－y(t)→0时的控制量，c1、c2是对y(t)变化的限制参数，且有c2≤c1。上述控制量及限制参数的大小可根据电阻炉功率大小再结合专家经验及现场调试来确定。
4　结束语
　　本系统以单片机为核心，由于采用了热偶放大器及光耦晶闸管，使硬件接口简单，软件编程简化；液晶显示模块的采用使提示信息量更为丰富,操作简便；运用了智能控制领域的产生式控制算法，使控温精度提高。在实际现场(4 kW电阻炉)应用中表明，控温精度＜±2 °C，温度测量范围为0～1300 °C，抗干扰能力强，运行可靠，各项技术指标满足国标GB212-77的要求。此外，本仪器还可做一般测温仪器使用，若对软件稍作修改，就可应用到其它测控温场合。
